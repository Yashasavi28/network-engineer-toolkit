About NetOpsKit

NetOpsKit is a free toolkit built for network engineers and NetOps teams. It provides fast networking calculators and configuration generators to help engineers work more efficiently.

Tools include subnet calculators, CIDR summarization, IPv6 tools, configuration generators and more.

NetOpsKit was created to simplify networking calculations and make common networking tasks faster.

Our goal is to build the most useful toolkit for network engineers worldwide.
